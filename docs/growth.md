### 🎯 Goal

Scale organic traffic from **0 → 1M monthly visits** using programmatic SEO + distribution loops.

---

## Phase 0 — Foundation (Week 0–2)

Targets:

* Buy domain + deploy Astro site
* Setup Search Console + Analytics
* Publish first **150–300 pages**

Execution:

* Focus on indexing, not traffic
* Submit sitemap manually
* Ensure internal linking graph exists

Expected:

* Very low traffic
* Some impressions

Targets:

* Reach **800–1200 pages**
* Cover 5 major writers
* Cover 5 major categories

Traffic expectation:

* 2k–10k visits/month

Execution tactics:

* Publish daily (content velocity signal)
* Create collection pages like:

  * 50 sad shayari

* Pinterest pin automation
* Reddit niche posting
